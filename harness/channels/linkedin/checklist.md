# LinkedIn 발행 전 체크리스트

## 자동 점검

- [ ] 본문 길이 ≤ 3000자, 권장 800~1500자
- [ ] 첫 3줄 합산 ≤ 220자 (피드 컷)
- [ ] 해시태그 3~5개
- [ ] `banned.words / topics / claims` 미포함
- [ ] 본문 첫 줄에 외부 링크 없음 (도달 페널티)
- [ ] (광고 분류 시) `legal.adHashtag` 또는 명시적 표기 포함
- [ ] `hashtags.always` 모두 포함
- [ ] 캐러셀 있는 경우: 첫 슬라이드에 핵심 메시지 + 9장 이내

## 수동 점검

- [ ] 첫 3줄이 "더 보기"를 누르게 만드는가?
- [ ] 마지막에 댓글 유도 질문이 있는가?
- [ ] 회사 톤(`tone.voiceNotes`)을 한 단계 professional 쪽으로 보정했는가?
- [ ] 자뻑·슬로건 톤이 아닌가?
- [ ] 외부 링크는 본문이 아닌 댓글에 두기로 했는가?
- [ ] 멘션이 정확한가? (`@회사`, `@사람` URN)
- [ ] 캐러셀의 텍스트가 모바일에서 읽히는가?

## 시리즈인 경우

- [ ] 1편이 단독으로도 가치가 있는가?
- [ ] 다음 편 예고가 자연스럽게 깔려 있는가?
