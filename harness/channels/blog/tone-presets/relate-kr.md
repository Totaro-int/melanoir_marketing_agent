# Tone Preset — relate.kr (정갈한 정중체)

> 기본 가이드 (`style-guide.md`) 와 동일. 학술/B2B/정보성 블로그 글에 권장.
> brand-guardian 자동 검증 — 친근체 혼용 시 warn, 광고법 위반 시 block.

## 핵심 특징

- 이모지 0
- bold (`**`) 최소화 (핵심 명제 1-3회만, 단락마다 X)
- 정중체 통일 (`~합니다`, `~할 수 있습니다`, `~됩니다`)
- 친근체 금지 (`~해요`, `~거든요`, `~잖아요`, `~네요`, `~죠`)
- 종결어미 다양화 — 같은 어미 3회 연속 X (`~합니다. ~합니다. ~합니다.` 패턴 금지)
- 단락 평균 2-3 문장 (7 문장 이상 단락 금지)
- `키워드 — 한 줄 설명` 대시 리스트
- 정량 수치 그대로 (`SPF 2.5`, `λc 388 nm`, `4,086자`)
- 추상 형용사 자제 (`효율적`, `다양한`, `혁신적`, `획기적`, `강력한`)
- 학술 인용 / 비포-애프터 수치
- FAQ: `Q.N` + `A.` 접두 + Q/A 사이 빈 줄
- AEO: H2/H3 / 표 / FAQ / 정량 수치 4개 중 최소 3개 본문 포함
- 자료 인용: sourceMaterials 있으면 본문 끝 `## 참고` 섹션 의무

## 어울리는 캠페인

- 학술 논문 정리
- B2B 인사이트
- 제품 데이터 / 시험성적서 정리
- 기술 블로그
- 정보성 콘텐츠 (영업 X)

## 어울리지 않는 캠페인

- 감성 일상 글
- 친밀한 톤 필요한 D2C 브랜드
- 빠른 행동 유도 (영업)

## 예시 단락

> 멜라누아 엠보는 필수 항목을 공인기관에서 시험해 All N.D.(불검출) 판정을 받았습니다. 같은 종류 색소들이 평균 3~5종 미량 검출되는 점을 고려하면 의미 있는 차이입니다.
