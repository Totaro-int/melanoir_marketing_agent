# Tone Preset — Sales (직접 영업 / 행동 유도)

> 제품 직접 소개 + 구매 유도. 광고법 (약사법, 표시광고법, 식약처) 준수 필수.
> brand-guardian 가장 엄격하게 자동 검증 — block 패턴 위반 시 발행 거부.
> sales 프리셋만 영업 톤 ("지금 ~", "한정") 일부 허용 — 광고법 block 패턴은 동일하게 차단.

## 핵심 특징

- 이모지 절제 사용 (강조 / 제목)
- bold 강조 적극 (핵심 메시지 / 가격 / 혜택)
- 정중체 (`~합니다`) + 일부 친근체 mix
- 첫 단락 = 핵심 가치 제안
- 구체적 가격 / 혜택 / 한정 / 보장 명시
- 비교 (`기존 제품 X` vs `우리 제품 Y`) — 단 비방 X
- 사용자 후기 / 사례 / 데이터
- CTA 명확 — `지금 ~`, `~ 문의하기`
- mustExclude — 의학적 효능, 100%, 부작용 없음, 치료, 완벽

## 어울리는 캠페인

- 제품 런칭
- 시즌 프로모션
- 한정 판매
- 신규 가입 / 첫 구매 유도

## 어울리지 않는 캠페인

- 학술 정리
- 객관적 비교 가이드
- AEO 정보성 글

## 예시 단락

> 멜라누아 엠보는 28종 유해물질 불검출, 무균 시험 적합, 자극 인자 51% 감소 데이터를 시험성적서로 보유합니다. 반영구 시술 아티스트가 색소 선택 시 가장 먼저 확인하는 안전 기준 세 가지를 모두 충족한 색소입니다. 시술 클레임 리스크를 줄이고 싶다면 시험성적서 PDF를 먼저 확인해보세요.

## 광고법 체크리스트

- 의학적 효능 주장 없음
- "100%", "유일한", "최고의" 추상 단정 없음
- 효과 보장 / 부작용 없음 단정 없음
- 가격 / 혜택 표시 시 단위 / 기간 명시
- 비교 광고는 객관적 수치만 (비방 X)
