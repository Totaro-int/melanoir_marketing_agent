# Tone Preset — Informational (정보성 / AEO 친화)

> 검색 / AI 브리핑 (Cue:, Perplexity) 친화. 정보 전달이 가장 큰 가치.

## 핵심 특징

- 이모지 0
- 정중체 통일 (`~합니다`)
- 객관적 / 중립적 톤 — 자기 브랜드 영업 X
- 첫 단락 = 한 문장 요약 (AEO 검색 결과 발췌 친화)
- H2 = 명확한 질문/답변 구조 (`X 란 무엇인가`, `왜 Y 인가`)
- H3 = 세부 항목 (구체적 / 검색 가능 키워드)
- FAQ 5개 (Q.N / A. 형식)
- 출처 명시 (DOI, 공인기관, 정부 가이드라인)
- 마무리 = 체크리스트 또는 요약 5점

## 어울리는 캠페인

- AEO 최적화 (Cue:, AI 검색)
- 학술 / 산업 지식 정리
- 규제 / 인증 / 안전 정보
- 비교 / 평가 가이드
- Wikipedia-like 정리 글

## 어울리지 않는 캠페인

- 영업 직접
- 감성 후기
- 단순 제품 소개

## 예시 단락

> 자외선 차단 성분은 메커니즘에 따라 세 가지로 분류됩니다. 화학 차단제는 자외선을 흡수해 열로 변환하고, 무기 차단제는 자외선을 반사하거나 산란시키며, 천연 멜라닌은 분자 내부 공액 결합 구조로 광역 흡수와 항산화를 동시에 수행합니다.

## AEO 최적화 추가 규칙 (brand-guardian 자동 검증)

- **두괄식** — 결론 먼저. 첫 단락 = 핵심 답변
- **한 H2 당 한 질문** — H2 헤딩 4-6개. 각 H2 가 검색 쿼리 형식
- **각 단락이 독립적** — 검색 결과 발췌 가능. 앞 단락 모르고 읽어도 의미 통함
- **정량 수치 5개 이상** — `%`, `nm`, `mg`, `회`, `장`, `일`, `주`, `개월` 단위 포함
- **구문 패턴** — `<주체> 는 <조건>일 때 <결과> 합니다.` 또는 `<X> 는 <Y> 입니다.` (직답 형식)
- **FAQ 섹션 의무** — `## 자주 묻는 질문` + Q.N/A. 형식, 3-5개 항목
- **표 또는 마무리 체크리스트** — 정량 비교/요약 표 1개 또는 체크리스트 5점 이하

## 자료 인용 (E-E-A-T 강화 필수)

informational 톤 글은 거의 항상 출처가 있어야 신뢰. brief.sourceMaterials.texts 에 자료 있으면 본문 끝 `## 참고` 또는 `## 출처` 섹션:

```markdown
## 참고

- Han, S. et al. (2021). Title. Journal Name, Vol(Issue), pp. DOI 또는 URL
- 식약처 가이드라인 — 화장품 광고 표시 (2023)
- 시험성적서 — 공인기관명, 시험번호
```

학술 자료 우선, 정부/공인기관, 자체 시험성적서 순. URL 가능하면 명시.

## 검색엔진 추가 점수 (선택)

- `> ` 인용 1-2개 (핵심 명제) — Google snippet 추출 친화
- 메타 키워드 — 첫 단락에 주요 키워드 1-2회 자연 등장
- H2 텍스트에 longtail 키워드 — `천연 자외선 차단 성분 비교` (단순 `비교` 보다 좋음)
