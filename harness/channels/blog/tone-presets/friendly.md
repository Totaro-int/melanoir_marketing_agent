# Tone Preset — Friendly (친근 D2C)

> 소비자 직접 영업 / 라이프스타일 / D2C 브랜드 일상 글에 권장.
> brand-guardian 자동 검증 — 친근체 어미 0개 = info (너무 딱딱), 광고법 위반 시 block.

## 핵심 특징

- 이모지 절제 사용 (글당 3-5개, 끝맺음 또는 강조에만 — 6개 초과 시 AI tone 의심)
- bold 자유롭게 사용 (한 단락에 1-2개)
- 친근체 (`~해요`, `~거든요`, `~잖아요`, `~네요`, `~죠`) 자연스럽게
- 정중체 (`~합니다`) 도 섞을 수 있음 — 시작/끝에 1-2회
- "여러분", "저희" 호칭 OK
- 단락 1-3 문장 (가벼움)
- 첫 단락 hook 강함 (`혹시 ~ 한 적 있으세요?` / `~ 다들 한번쯤 고민하시죠?`)
- 정량 수치 + 일상 비유 (`SPF 50 = 자외선 1/50 통과`)
- 마무리에 행동 유도 (`~ 한번 해보세요`, `궁금하면 댓글로`)
- 광고법 — D2C 라도 광고법 적용. 의학적 효능/100% 단정 절대 금지
- AEO 약화 OK — friendly 톤은 검색보다 후기/공감 중심

## 어울리는 캠페인

- 스킨케어 일상
- 라이프스타일 후기
- 제품 사용기
- 음식 / 패션 / 여행

## 어울리지 않는 캠페인

- 학술 인용
- 의학적 효능 주장 (광고법)
- B2B 견적 / 계약 안내

## 예시 단락

> 혹시 선크림 발랐는데도 햇볕만 보면 트러블 나는 적 있으세요? 저도 그랬거든요. 알고 보니 화학 차단제 성분에 민감한 피부였어요. 요즘은 천연 멜라닌 같은 자연 유래 차단 성분이 떠오르고 있는데, 한번 살펴볼까요?
