# Tone Preset — B2B (인사이트 / 의사결정자)

> 기업 의사결정자 / 구매 담당자 / 파트너십 대상.

## 핵심 특징

- 이모지 0
- 정중체 통일 (`~합니다`)
- 영문 키워드 자연스럽게 (`MoQ`, `OEM`, `B2B`, `ROI`, `Lead time`)
- 데이터 / 비교 / 사례 위주 (감성 X)
- 단락 3-5 문장 (정보 밀도 높음)
- "구매팀이 자주 묻는 질문" 같은 페르소나 명시
- 가격 / 조건 / 인증 / 규제 명시
- 마무리 — 연락처 / 견적 요청 CTA

## 어울리는 캠페인

- LinkedIn 인사이트
- 산업 리포트
- 제품 카탈로그 / 카탈로그 비교
- 무역 / 수출입 안내
- 인증 / 규제 정리

## 어울리지 않는 캠페인

- 소비자 일상 글
- 감성 / 라이프스타일

## 예시 단락

> K-뷰티 OEM/ODM 진입 시 일반적인 MoQ 는 1,000-3,000 pcs 입니다. CGMP 인증 공장 기준 평균 lead time 은 8-12주이며, 자체 브랜드 OEM 전환 시 ISO22716 추가 인증이 요구됩니다. 본 글은 50개 이상 한국 OEM 사 운영 데이터를 기반으로 합니다.
