# Tone Preset — B2B (인사이트 / 의사결정자)

> 기업 의사결정자 / 구매 담당자 / 파트너십 대상.
> brand-guardian 자동 검증 — 친근체 1회 이상이면 warn, 광고법 위반 시 block.

## 핵심 특징

- 이모지 0
- 정중체 통일 (`~합니다`)
- 친근체 절대 금지 — B2B 신뢰감 직접 깎음
- 영문 키워드 자연스럽게 (`MoQ`, `OEM`, `B2B`, `ROI`, `Lead time`, `CGMP`, `ISO22716`)
- 약어 첫 등장 시 풀이 (`MoQ (Minimum Order Quantity)`) — 1회만
- 데이터 / 비교 / 사례 위주 (감성 X)
- 단락 3-5 문장 (정보 밀도 높음)
- "구매팀이 자주 묻는 질문" 같은 페르소나 명시
- 가격 / 조건 / 인증 / 규제 명시
- 마무리 — 연락처 / 견적 요청 CTA
- 영업 톤 자제 — "지금 바로", "한정", "놓치지" 사용 X (B2B 는 의사결정 사이클이 길어서 의미 X)
- AEO 친화 — 표/체크리스트 적극 활용 (B2B 검색은 비교 데이터 검색이 많음)

## 어울리는 캠페인

- LinkedIn 인사이트
- 산업 리포트
- 제품 카탈로그 / 카탈로그 비교
- 무역 / 수출입 안내
- 인증 / 규제 정리

## 어울리지 않는 캠페인

- 소비자 일상 글
- 감성 / 라이프스타일

## 예시 단락

> K-뷰티 OEM/ODM 진입 시 일반적인 MoQ 는 1,000-3,000 pcs 입니다. CGMP 인증 공장 기준 평균 lead time 은 8-12주이며, 자체 브랜드 OEM 전환 시 ISO22716 추가 인증이 요구됩니다. 본 글은 50개 이상 한국 OEM 사 운영 데이터를 기반으로 합니다.
