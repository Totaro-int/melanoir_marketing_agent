# Instagram Post Templates

`copywriter` + `image-director` 서브에이전트가 사용할 카피·카드 템플릿. 각 템플릿은 **구조 가이드**일 뿐, 그대로 채워 넣지 말 것 — 회사 프로필의 톤·시각으로 재해석한다.

> 인스타그램은 **비주얼 first** 매체. 카드뉴스 디자인 의사결정이 캡션보다 더 중요. cardVisual 가이드는 image-director.md 참조.

---

## I1. 단일 인사이트형 (1장 카드, 캡션 100~250자)

**카드 (image-director)**: Stat Card — 어두운 배경 + Hero stat + eyebrow + 1줄 caption  
**캡션 (copywriter)**:

```
{후킹 1줄: 카드의 메시지를 풀어쓰는 첫 문장}

{관점 1~2줄: 왜 그게 의미 있는지}

{전환 1줄: 그래서 무엇이 달라지는가}

{CTA 1줄: 프로필 링크 안내}

{해시태그 5~15개}
```

**예시 (cos.totaro 톤)**:
```
출장 한 번에 4시간이 사라집니다. 한국 화장품 OEM 후보 좁히는 작업 얘기예요.

토타로 코스는 그 4시간을 3초로 줄였습니다. 1만 5천 곳의 인증·MOQ·리드타임을
한 양식으로 정규화한 결과예요.

지금 신규 라인 검토 중이시라면 프로필 링크에서 첫 매칭은 무료로 확인해보세요.

#K뷰티 #한국화장품 #B2B소싱 #OEM매칭 #코스메틱OEM #TOTARO
```

## I2. 케이스 스터디 캐러셀 (3장, `series-3`)

**카드 1 — hook (P+E)**: Stat Card — 핵심 결과 수치 거대화  
**카드 2 — body (S+O)**: Split Layout — 좌 텍스트 / 우 다이어그램·시각  
**카드 3 — cta (N+A)**: Carousel-friendly — 행동 유도 단순 디자인

**캡션 (copywriter, 200~400자)**:

```
{첫 125자에 핵심 결과 수치 + 맥락}

{본론 2~3줄: 어떻게 그 결과가 나왔는지}

{CTA 1줄: 프로필 링크}

{해시태그 8~12개}
```

## I3. 가이드·교육 캐러셀 (5~10장, `series-5`)

**카드 1 — hook (P+E)**: 어떤 문제를 푸는지 한 화면 요약  
**카드 2~4 — body (S+O 단계)**: 가이드 단계별. 카드당 1개 토픽  
**카드 5 — cta (N+A)**: 정리 + 다음 단계 안내

**카드 디자인 일관성**:
- 카드별 페이지 인디케이터 (1/5, 2/5, ...) 우상단
- 동일 폰트·여백·아이콘 스타일
- 색은 카드별 accent 변경 가능 (단, primary/background는 동일)

**캡션 (copywriter, 300~500자)**:

```
{첫 125자: 가이드 주제 + 누구를 위한 것인지}

{본론 3~5줄: 카드뉴스에서 다루는 단계 요약}

{CTA 1줄: 더 깊은 자료 또는 프로필 링크}

{해시태그 10~15개}
```

## I4. 제품 런칭 캐러셀 (3~5장)

**카드 1 — hook**: 런칭 메시지 (드라마틱한 도입 OK — `goal: launch`)  
**카드 2 — body**: 핵심 기능 2~3개  
**카드 3~4 — body**: 차별점 / 사례 / 수치  
**카드 마지막 — cta**: 출시 일정 + 행동 유도

**캡션**:
- launch goal이라 P 약하게, S+O 강하게
- "드디어" 같은 단어 활용 가능 (단, 톤 보존)

## I5. 비포·애프터 카드 (2장, `series-2`)

**카드 1**: Before — 문제 상황 시각화  
**카드 2**: After — 해결된 상태 시각화  
**디자인 일관**: 같은 구조·다른 색조 (Before 회색조 / After 브랜드 색)

**캡션 (copywriter, 100~200자)**:

```
{첫 125자: 변화 핵심을 한 줄로}

{보조 1~2줄: 어떻게 가능했는지}

{CTA 1줄}

{해시태그 5~10개}
```

## 채우기 규칙 (copywriter용)

1. `{...}` 자리는 회사 프로필의 톤으로 재작성. 템플릿 문장을 그대로 두지 말 것.
2. `tone.sampleSentences`가 있으면 그 호흡을 따라할 것.
3. **첫 125자에 핵심 메시지** (피드 미리보기 컷).
4. 해시태그는 본문 끝에 줄바꿈 후, **5~15개**.
5. 캐러셀 카드 시각은 image-director에게 위임. 카피라이터는 `cardVisual` 작성 X (image-director가 담당).

## 채우기 규칙 (image-director용)

1. 카드 레이아웃은 **카드 레이아웃 카탈로그 5종** 중 선택 (image-director.md 참조).
2. 채널 = instagram → **어두운 배경 + Glow B 강하게**.
3. Hero stat 120px 이상, Eyebrow 18~22px, Caption 26~32px.
4. 시리즈인 경우 카드별 디자인 일관 (페이지 인디케이터, 폰트, 여백).
5. brand color는 `imageContext.visual.colors` 에서만 — 레퍼런스 색 차용 X.
