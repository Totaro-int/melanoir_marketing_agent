# Instagram 발행 전 체크리스트

휴먼 승인 게이트(`/sns-preview <slug>` → approve)에서 사용. `copywriter` + `image-director` 산출물에 대해 자동 + 수동 점검.

## 자동 점검 (Phase 3 게이트가 차단)

### 캡션 (copywriter 산출물)
- [ ] 본문 길이 ≤ 2,200자
- [ ] **첫 125자 (피드 미리보기)**에 핵심 메시지 포함
- [ ] 해시태그 개수: 5~15개 (한도 30 이하)
- [ ] `banned.words` 미포함
- [ ] `banned.claims` 미포함 ("최고의", "1위", "유일한", "100%" 등)
- [ ] `banned.topics` 미포함
- [ ] 외부 링크 직접 노출 X (대신 "프로필 링크에서" 표현)
- [ ] `hashtags.always`가 모두 포함됨
- [ ] (광고 분류 시) `legal.adHashtag` 포함

### 카드 (image-director 산출물)
- [ ] 카드 1~10장 범위
- [ ] 모든 카드 1:1 또는 4:5 aspect (1080×1080 또는 1080×1350)
- [ ] 첫 카드(hook)에 Hero 요소 96px 이상
- [ ] 첫 카드 텍스트 점유 70% 이하 (피드 가독성)
- [ ] 모든 카드에 `imageContext.visual.colors` 사용 (브랜드 색)
- [ ] 어두운 배경 카드면 `glow B` 또는 그에 준하는 깊이감 적용
- [ ] 시리즈인 경우 카드 디자인 톤·폰트·색 일관

## 수동 점검 (사람이 확인)

### 비주얼
- [ ] 첫 카드만 봐도 캐러셀을 펼치고 싶은가?
- [ ] 카드뉴스 디자인이 회사 `imageStyle.aestheticDirection`에 맞는가?
- [ ] 모바일에서 본 텍스트 가독성 (실제 디바이스 또는 1080px scale 미리보기)?
- [ ] 카드 마지막에 brand 표기 (로고·텍스트) 자연스럽게 들어갔는가?

### 카피
- [ ] 첫 125자가 스크롤을 멈출 만큼 흥미로운가?
- [ ] 회사 톤(`tone.voiceNotes`)과 일치하는가?
- [ ] 광고 카피처럼 들리지 않는가? (느낌표·"지금 바로" 류 회피)
- [ ] 사실 확인 — 숫자·인용·날짜
- [ ] CTA가 "프로필 링크에서" 표현으로 자연스럽게 들어갔는가?

### 시리즈/캐러셀인 경우 추가
- [ ] 첫 카드(hook)가 단독으로도 의미가 통하는가?
- [ ] 카드별 텍스트 양·디자인 톤이 일관적인가?
- [ ] 마지막 카드(cta)에 명확한 행동 제안 1개 (강요 X)

## 실패 시 행동

- 자동 점검 실패: 사용자에게 어느 룰이 깨졌는지 표시 + 자동 수정 제안
- 수동 점검에서 사용자가 reject: 사용자가 적은 사유를 `copywriter` 또는 `image-director`에 피드백으로 전달해 재생성 (`generate.mjs --partial=N`)
