# Brunch Post Template

> 공통 정보성 글 템플릿 5종(B1~B5)은 [`../../blog/templates/post.md`](../../blog/templates/post.md) 사용.  
> 본 문서는 브런치 매체 특수 가이드만.

## 브런치 추가 채우기 규칙

위 공통 템플릿 + 다음 추가 사항 적용:

1. **시적·관찰 톤 우선** — "방법" "정리" "가이드" 같은 SEO 키워드 X
2. **1인칭 사용** ("저는", "우리는") — 보고체 X
3. **매거진 지정 필수** — front-matter에 `magazine: "<매거진명>"`
4. **태그 5개 한도** — long-tail 위주
5. **헤더 사진 1장 + 본문 1~3장** — 인물·풍경·분위기 OK
6. **마지막 disclaimer X** — 브런치는 작가의 글
7. **글자수 1,500~3,000자** (자유, 길어도 OK)
8. **Front-matter 형식**:
   ```
   ---
   title: "<시적 제목 — SEO 키워드 X>"
   tags: ["tag1", "tag2", "tag3", "tag4", "tag5"]
   magazine: "<매거진명>"
   ---
   
   <본문 markdown — 사진 인라인>
   ```

## 추천 글 종류 (브런치 적합)

- **B4 인사이트·관점형** — 가장 적합 (관찰·통찰 우선)
- **B2 케이스 스터디** — 인물 사진 풍부 가능
- **B1 가이드형 / B5 제품 소개형** — 톤을 에디토리얼로 변환 시 가능

비추천:
- 직접 광고형 (브런치는 즉시 거부)
- SEO 키워드 도배형
- 디렉터리·리스트형
