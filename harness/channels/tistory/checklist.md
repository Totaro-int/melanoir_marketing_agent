# Tistory 발행 전 체크리스트

> 공통 점검은 [`../blog/checklist.md`](../blog/checklist.md) 참조.  
> 본 문서는 티스토리 매체 특수 점검만.

## 자동 점검 (티스토리 추가)

### 메타·태그
- [ ] 태그 5~10개 (티스토리 한도 10)
- [ ] 카테고리 지정 (URL 영향)
- [ ] URL slug과 콘텐츠 키워드 일치 (잡블로그 회피)

### 내부 링크 (티스토리 핵심)
- [ ] 내부 링크 **2~4개 필수** (체류시간 = 가중치)
- [ ] "이전 글에서 정리" 류 자연 삽입 표현

### 구조
- [ ] H1은 제목만 사용 (본문 H1 절대 금지)
- [ ] H2 4~6개, 각 섹션 첫 줄 결론

## 수동 점검 (티스토리 추가)

- [ ] 다음·구글 검색 양쪽에서 노출 가능한 제목인가?
- [ ] 외부 트래픽(네이버·SNS) 유입 가능성 있는 주제인가?
- [ ] 광고 자연스럽게 녹였는가? (도배 X)
- [ ] 카테고리 일관성 (잡블로그 신호 X)
