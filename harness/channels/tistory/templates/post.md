# Tistory Post Template

> 공통 정보성 글 템플릿 5종(B1~B5)은 [`../../blog/templates/post.md`](../../blog/templates/post.md) 사용.  
> 본 문서는 티스토리 매체 특수 가이드만.

## 티스토리 추가 채우기 규칙

위 공통 템플릿 + 다음 추가 사항 적용:

1. **태그 5~10개** (티스토리 한도 10)
2. **내부 링크 2~4개 강제** — 체류시간이 다음·구글 가중치 핵심
3. **카테고리 지정** — URL 슬러그과 일치하게
4. **Visibility = 3** (공개)
5. **첫 줄에 외부 링크 절대 금지** — 다음 알고리즘 페널티
6. **Front-matter 형식** (browser-publish 가 파싱):
   ```
   ---
   title: "<제목>"
   tags: ["tag1", "tag2", ...]
   category: "<카테고리명 또는 ID>"
   ---
   
   <본문 markdown>
   ```

## 광고 도배 회피

- 본문 안 광고 1~2회만 자연 삽입
- 결론 섹션에 soft CTA 1개
- "지금 클릭!", "한정 특가" 류 직접 광고 금지 → 다음·구글 페널티
