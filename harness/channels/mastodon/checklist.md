# Mastodon 발행 전 체크리스트

## 자동 점검

### 텍스트
- [ ] 본문 500자 이하 (인스턴스 한도 확인)
- [ ] 한국어는 ~400자 이하 권장
- [ ] 해시태그 2~5개 (검색 의존)
- [ ] `banned.words / claims / topics` 미포함
- [ ] (광고 분류 시) `legal.adHashtag` 포함

### 이미지
- [ ] 0~4장 범위
- [ ] 1:1 또는 16:9 aspect
- [ ] (선택) ALT 텍스트 권장 (Mastodon 접근성)

### 봇 패턴 회피
- [ ] 정확한 시간 발행 X (분 단위 변동)
- [ ] 정형 머리말 X
- [ ] AI 정형 어휘 X

## 수동 점검
- [ ] 인스턴스 문화에 맞는 톤인가?
- [ ] 답글 응대 가능한 시간대인가?
- [ ] CW 필요한 토픽인가? (긴 글·민감 주제)

## 실패 시
- 자동 실패: 어느 룰 깨졌는지 표시
- 수동 reject: copywriter 재생성
