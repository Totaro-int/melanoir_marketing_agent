# Mastodon Post Templates

`copywriter`가 사용. **500자 + 해시태그 2~5개 + 사람 목소리 + 인스턴스 문화 존중**.

---

## M1. 짧은 관점 (~200자, 이미지 0)

```
{단정 또는 관찰 1줄.}

{보조 1~2줄}

#태그 #2~5개
```

---

## M2. 케이스 스터디 (400~500자 + 1~2장)

```
{결과 수치 또는 변화 1줄}

{배경·맥락 1~2줄}

{어떻게·왜 1~2줄}

#태그 #2~5개
```

---

## M3. Thread (3~5편, 답글 체인)

**1편 (티저, ~250자)**: 결론 살짝 + thread 시작 표시

**2~N편 (답글, 각 ~250자)**: 단계·근거·맥락

**마지막 편**: 정리 + 가벼운 링크

---

## M4. 긴 글 (CW 활용, ~500자)

```
CW: {경고 라벨 — 토픽}
---
{긴 본문 — 500자 안}

#태그
```

> CW(Content Warning)는 길거나 민감한 토픽에 활용. 광고성 콘텐츠는 CW 써도 거부.

---

## 채우기 규칙

1. **500자 한도** (인스턴스 한도 확인 — 일부 5,000자)
2. **사람 목소리 우선** — 봇 패턴 X
3. **해시태그 2~5개** (검색 의존)
4. 인스턴스 문화 존중 (mastodon.social 일반 / fosstodon.org 오픈소스 / hachyderm.io tech)
5. 답글 응대 가능 시간대 발행
6. 이모지 도배 X
