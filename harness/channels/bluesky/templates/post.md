# Bluesky Post Templates

`copywriter` + `image-director`가 사용. **300자 한도 + 캐주얼·진솔 톤 + 광고 X**.

---

## S1. 짧은 관점 (단일, ~150자, 이미지 0)

**가장 자주 쓰는 형식. 짧을수록 잘 받음.**

```
{단정 또는 의외의 관점 1줄.}

{보조 1~2줄 (왜 그렇게 보는지·짧은 경험·짧은 통찰)}
```

**예시 (cos.totaro 톤)**:
```
한국 화장품 OEM 찾는 데 한 달씩 쓰는 게 당연한 일은 아니더라고요.

조건 한 줄 입력으로 3초 매칭, 그게 가능해진 게 데이터 정규화 덕분이에요.
```

---

## S2. 이미지 첨부 한 줄 (~100자 + 1~2장)

**이미지가 본문보다 강할 때.**

**카드 (image-director)**: 캐주얼·소셜 톤 1장 (1:1 또는 16:9)  
**텍스트**:

```
{이미지의 맥락 1~2줄}

{선택: 짧은 코멘트}
```

---

## S3. 시리즈 thread (3편)

**1편 (티저, ~150자)**: 결론을 살짝 던지기

**2편 (본문, ~150자, 답글로)**: 이유·데이터·관점

**3편 (마무리, ~150자, 답글로)**: 그래서 뭐 했는지 + 가벼운 링크

> 각 편이 단독으로도 의미 있어야 함 — 답글 안 펴는 사용자 다수.

---

## S4. 솔직 후기·발견 (~200자 + 스크린샷)

**개발자·디자이너 친화. Bluesky가 가장 잘 받는 톤.**

```
{발견·후기 1~2줄}

{왜 흥미로운지 1줄}

{선택: 짧은 코드·스크린샷·링크}
```

---

## 채우기 규칙

1. **300자 한도 절대 준수** (한국어 150자 권장)
2. **캐주얼 친근형 어미** (`~어요`/`~이에요`/`~더라고요`/`~잖아요` 섞기)
3. **첫 1~2줄에서 결판** — 길게 빌드업 X
4. **광고 톤·과장 절대 X** — "지금 신청" "한정" 즉시 외면
5. **봇 패턴 X** — 이모지 도배·정형 머리말 X
6. 해시태그 0~2개 (always 1개만 또는 생략)
7. `image-director`: 채널 = bluesky → 자유 비율, 캐주얼·친근, 인물 OK
