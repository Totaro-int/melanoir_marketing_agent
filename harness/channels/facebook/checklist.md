# Facebook 발행 전 체크리스트

## 자동 점검

### 캡션
- [ ] 본문 길이 250~600자 (한국어 100~250자)
- [ ] **첫 250자**(미리보기 컷)에 핵심 메시지
- [ ] 해시태그 2~5개
- [ ] 외부 링크 1개 이하 (도배 금지)
- [ ] `banned.words / claims / topics` 미포함
- [ ] `hashtags.always` 모두 포함
- [ ] (광고 분류 시) `legal.adHashtag` 포함

### 이미지·캐러셀
- [ ] 1~10장 범위
- [ ] 1:1 또는 16:9 aspect
- [ ] `imageContext.visual.colors` 적용
- [ ] 캐러셀이면 카드 디자인 일관

## 수동 점검
- [ ] 댓글 유도 질문 마지막에 1줄 있는가?
- [ ] 회사 톤(`tone.voiceNotes`)과 일치, 너무 격식·너무 캐주얼 X?
- [ ] 광고 외침 없는가?
- [ ] 사실 확인 — 숫자·날짜·인용

## 실패 시
- 자동 실패: 어느 룰 깨졌는지 표시 + 자동 수정 제안
- 수동 reject: 사유를 copywriter에 피드백 → 재생성
