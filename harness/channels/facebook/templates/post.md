# Facebook Post Templates

`copywriter` + `image-director`가 사용. 구조 가이드 — 그대로 채우지 말고 회사 프로필 톤으로 재해석.

---

## F1. 인사이트 (단일, 250~400자)

```
{후킹 1줄: 의외의 단정 또는 짧은 숫자}

{관점 2~3줄: 왜 그런지 + 우리 경험·데이터}

{전환 1줄: 그래서 무엇을 의미하나}

{댓글 유도 질문 1줄: "여러분은 어떠신가요?" 류}

#태그2~5개
```

---

## F2. 케이스 스터디 캐러셀 (3~5장 + 400~600자)

**캐러셀 카드 구조**:
- 카드 1: hook (결과 수치)
- 카드 2-3: 변화 시도 (Before/After)
- 카드 4: 데이터 차트
- 카드 5: 사례·CTA

**캡션**:
```
{첫 250자에 결과 + 맥락}

{본론 2~3줄: 어떻게 그 결과 나왔는지}

{CTA + 댓글 유도}

#태그2~5개
```

---

## F3. 가이드·체크리스트 (5~10장 캐러셀 + 500~700자)

**카드 디자인 일관**:
- 카드별 페이지 인디케이터 (1/7, 2/7...)
- 동일 폰트·여백·아이콘
- 친근 톤 (인스타보다 부드럽게)

---

## F4. 제품·이벤트 (200~400자 + 1~3장)

```
{핵심 한 줄: 무엇이·언제}

{왜 의미 있는지 1~2줄}

{어떻게 시작·참여하는지 1줄}

{링크 또는 댓글 유도}

#태그2~5개
```

---

## F5. 채용 (400~500자 + 분위기 사진)

```
{직무 + 매력 한 줄}

{회사·팀 소개 1~2줄}

{어떤 사람을 찾는지 1~2줄}

{지원 링크 1줄}

#태그2~3개 (#채용 등)
```

---

## 채우기 규칙

1. `{...}` 자리는 회사 프로필 톤으로 재작성
2. 첫 250자에 핵심 (미리보기 컷)
3. 댓글 유도 질문 마지막에 1줄 (알고리즘 가중치)
4. `image-director`: 채널 = facebook → 1:1 또는 16:9, 친근 톤, 인물 OK
