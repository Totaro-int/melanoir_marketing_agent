# X (Twitter) Post Templates

`copywriter` + `image-director`가 사용. **280자 안에 결판 + 짧을수록 강함**.

---

## X1. 짧은 단정 (~100자, 이미지 없음)

```
{단정 또는 의외의 짧은 통찰. 한 문장.}

{선택: 짧은 보조 1줄.}
```

**예시 (cos.totaro 톤)**:
```
출장 4시간이 검색 3초로 줄어든다.

K-뷰티 OEM 매칭, 데이터 정규화의 결과.
```

---

## X2. 수치 + 이미지 (~150자 + 1장)

**이미지**: Stat Card — Hero stat 150~200px + 작은 caption

**텍스트**:
```
{수치 1개 + 맥락 1줄.}

{선택: 링크 1개 또는 #태그 1~2개}
```

**예시**:
```
5.2일 → 1.8일.

이 차이가 정산 인프라가 만들어낸 결과입니다.
```

---

## X3. Thread 시리즈 (각 ~150자 × 5~10편)

**1편 (단독으로도 가치 있어야)**:
```
{충분히 흥미로운 단정 또는 결과 한 줄.}

{thread 시작 표시 — "🧵" 또는 "1/" 권장 X. 자연스럽게 다음으로}
```

**2~N편 (각 짧게)**:
```
{단계·근거·맥락 한 줄씩.}
```

**마지막 편 (CTA)**:
```
{정리 + 행동 유도 1줄.}

{링크 1개}
```

---

## X4. 제품 업데이트 (~200자 + 스크린샷)

```
{기능 한 줄: 무엇이 새로 가능한가}

{왜 필요했는지 1줄}

{도큐먼트 또는 사용 링크}
```

---

## 채우기 규칙

1. **280자 한도 절대 준수** (한국어 140자 권장)
2. 형용사 모두 제거 — 수치·사실만
3. 줄바꿈 1~2개 이하
4. 해시태그 1~2개만
5. 첫 트윗에 외부 링크 자제 (영상·이미지가 도달 ↑)
6. `image-director`: 채널 = x → 16:9 + 극도 미니멀 + Hero 거대화
