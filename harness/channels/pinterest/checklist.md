# Pinterest 발행 전 체크리스트

## 자동 점검

### 텍스트
- [ ] 핀 제목 100자 이하 (한국어 25~50자 권장)
- [ ] 설명 500자 이하 (한국어 100~250자 권장)
- [ ] 핵심 키워드 설명에 자연 분포
- [ ] 해시태그 0~4개
- [ ] **링크 필수** (핀이 외부 트래픽 매체)
- [ ] 보드 ID 지정 필수 (`pinterest.boardId`)
- [ ] `banned.words / claims / topics` 미포함

### 이미지
- [ ] **세로형 2:3 (1000×1500)** 권장
- [ ] 이미지 1장만 (캐러셀 X)
- [ ] 텍스트 오버레이 영역 확보 (선택)
- [ ] 영업·광고 톤 X (Pinterest 거부)

## 수동 점검
- [ ] 보드 일관성 — 같은 보드의 다른 핀과 톤·주제 일치?
- [ ] 모바일 피드에서 시선 끌기 (세로 길이 활용)
- [ ] 검색 키워드 자연스러운가? (stuffing X)

## 실패 시
- 자동 실패: 어느 룰 깨졌는지 표시
- 수동 reject: copywriter 재생성 (Pinterest SEO·시각 톤 강조)
