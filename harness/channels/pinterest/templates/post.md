# Pinterest Pin Templates

`copywriter` + `image-director`가 사용. **이미지가 80% + 세로 2:3 + SEO 친화 설명**.

---

## P1. 인포그래픽 (텍스트 + 시각)

**카드 (image-director)**:
- 2:3 세로 (1000×1500)
- 인포그래픽 톤 (단계·통계·비교)
- **텍스트 오버레이 OK** — 핀 제목 큰 폰트로 상단 또는 하단 30% 영역
- 밝은 배경 + 브랜드 accent

**핀 제목** (25~50자):
```
{핵심 키워드 + 결과 약속}
예: "K-뷰티 OEM 매칭, 3초 안에 — 가이드"
```

**설명** (100~250자):
```
{문제 환기 1줄}
{솔루션·방법 1~2줄}
{관련 키워드 자연 분포}

#태그 0~4개
```

---

## P2. 무드보드·스타일

**카드**: 분위기 사진 (브랜드 톤·계절·라이프스타일)

**핀 제목** (시적, 짧게):
```
{시적 제목 — 분위기 키워드}
예: "초록 봄, K-뷰티 디스커버리"
```

**설명** (100~150자):
```
{분위기 묘사 1~2줄}
{선택: 출처·브랜드 언급}
```

---

## P3. 단계 가이드 (DIY/방법형)

**카드**: 단계별 시각화 (1·2·3 단계 표시)

**핀 제목**:
```
{대상} {결과 약속}하는 방법
예: "한국 화장품 OEM 매칭하는 5단계"
```

**설명** (200~400자):
```
{문제 환기 1줄}

{단계 요약}
- 1단계: ...
- 2단계: ...
- 3단계: ...

{외부 링크는 별도 필드 — 본문 X}

#태그
```

---

## P4. 제품 컬렉션

**카드**: 제품 클리어 사진 (배경 단순)

**핀 제목**: 제품명 + 카테고리

**설명**:
```
{제품 한 줄 소개}
{핵심 특징 1~2줄}
{카테고리·키워드 자연}

#태그
```

---

## P5. 비포·애프터

**카드**: 분할 비교 이미지 (좌 Before / 우 After)

**핀 제목**: "Before & After: {변화 키워드}"

**설명** (150~250자):
```
{변화 핵심 한 줄}
{어떻게 가능했는지 1~2줄}
{관련 키워드}

#태그
```

---

## 채우기 규칙

1. **세로 2:3 절대 권장** (모바일 피드 노출 최적)
2. **핀 제목 25~50자** + **설명 100~250자**
3. **링크 필수** (별도 필드 — 본문에 외부 URL 넣지 말 것)
4. **보드 지정 필수** + 보드 일관성
5. **이미지 안 텍스트 오버레이 OK** (다른 채널과 다름)
6. **영업·광고 톤 X** (Pinterest 거부)
7. `image-director`: 채널 = pinterest → 2:3 세로 + 인포그래픽·무드보드 톤
